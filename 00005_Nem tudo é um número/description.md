Mas os operadores não servem apenas para obter números a partir de outros, também podemos compará-los entre si! :muscle:

> Tente o seguinte no console:
>
> ``` python
ム10 > 9
```
>
> ``` python
ム11 >= 11
```
>
> ``` python
ム87 < 87
```
>
> ``` python
ム87 <= 100
```
>
> ``` python
ム9 == 9
```
>
> ``` python
ム8 != 8
```
> O que você acha que retornará? Um número? :thinking:
